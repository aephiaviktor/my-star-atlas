MY STAR ATLAS — WINDOWS PORTABLE

1. Extract the entire ZIP to a normal folder. Do not run it from inside the ZIP.
2. Double-click create-desktop-shortcut.vbs once.
3. Start My Star Atlas from the new desktop shortcut, or double-click launch-my-star-atlas.vbs.
4. Open Settings and enter your own connection/profile details.
5. Use the Update button inside the app for future versions.

Your settings are stored outside this application folder and are not included in the ZIP.
